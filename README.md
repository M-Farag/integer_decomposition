# Integer Decomposition
Simple algorithm to solve the integer decomposition challenge {Algebra 1}

### Specifications
- A RUST Application.
- Unisinged 32bit integer.

### How to use
- Pull the repo

1)
  - Run ``` cargo run ```

2) OR
  - Run ``` cargo build --release ```
  - Navigate to: /target/release
  - Run ``` ./int_decomposition ```
